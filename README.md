# NEON GUIDE — Tokyo Nightlife Intel

Community-powered nightlife review platform for international travelers in Tokyo.

## Features
- Spot listings with business account images (banner-style)
- Review system with category ratings
- Promoted shop pages (Staff, Events, Coupons)
- Community bulletin board (LFG, Q&A, Tips, Warnings)
- Staff showcase with photo profiles
- Mobile-first responsive design
- Admin panel for test data management

## Deploy
This is a static single-page app. Just enable GitHub Pages on `main` branch.
